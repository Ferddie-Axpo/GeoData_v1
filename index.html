📊 About This Calculator

This application calculates distances between the first 3 cities in each continent using the Haversine formula, which computes the great-circle distance between two points on a sphere given their latitudes and longitudes.

Features:

  • Calculates distances between all city pairs in each continent (3 cities = 3 distances per continent)
  • Uses real geographical coordinates for accurate distance measurements
  • Provides continental statistics and distance analytics
  • Displays results in both kilometers and miles
